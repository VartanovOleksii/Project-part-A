## Опис Класів та Поведінка

### Клас `Artist`

* **Основні дані:** Ім'я, Вік, Зарплатня, Інструмент, Активність, Кількість фанатів.
* **Валідація:** Ім'я (3-50 символів), Вік (18-95), Зарплатня (> 0), Інструмент (3-50, латинські символи/цифри), Кількість фанатів ($\ge 0$).

### Клас `Band`

* **Основні дані:** Назва, Кількість прослуховувань за місяць, Продюсери, Артисти.
* **Валідація:** Назва (3-50 символів), Кількість прослуховувань за місяць ($\ge 0$).

### Клас `Producer`

* **Основні дані:** Ім'я, Вік, Зарплатня, Спеціалізація, Стаж.
* **Валідація:** Ім'я (3-50 символів), Вік (20-95), Зарплатня (> 0), Спеціалізація (3-50 символів), Стаж (3-90).

### Клас `RecordLabel`

* **Основні дані:** Назва (3-50 символів), Групи.
* **Поведінка:** Додати групу, Відписати групу.

### Клас `Song`

* **Основні дані:** Назва (3-50 символів), Кількість прослуховувань ($\ge 0$), Група, Жанр.
* **Поведінка:** Запустити, Зупинити, Вивести метадані.
